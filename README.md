![Cloud](https://img.shields.io/badge/Cloud-AWS-FF9900?logo=amazonaws&logoColor=white)

This repository contains hands-on AWS demonstrations covering core cloud services and Infrastructure as Code (IaC).

Each demo focuses on a single AWS concept and is designed to be practical, repeatable, and easy to understand through real deployments, diagrams, screenshots, and walkthroughs.

The goal is to build cloud knowledge by provisioning and managing AWS infrastructure through code rather than manual configuration.

---

## Learning Objectives

By working through these demos, I aim to:

- Learn AWS through practical deployments
- Understand Infrastructure as Code using AWS CloudFormation
- Automate cloud infrastructure
- Apply AWS best practices
- Build repeatable cloud environments
- Gain hands-on experience with AWS services
